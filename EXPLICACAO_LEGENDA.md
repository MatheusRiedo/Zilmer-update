# 📝 Explicação: Como Adicionar Legendas

## O que é "Array"?

**Array** = Lista de várias coisas em ordem

### Exemplo do dia a dia:
Imagine uma lista de compras:
```
1. Leite
2. Pão
3. Ovos
```

No computador, isso é um "array" - uma lista ordenada.

---

## Qual Campo Usar?

### ✅ Use "Legenda da Imagem Principal" se:
- Você tem **APENAS 1 IMAGEM** na página
- É mais simples e direto
- Exemplo: Você vê só uma foto grande na página

**Como usar:**
1. Selecione a área (ex: Transporte)
2. Clique em **"Legenda da Imagem Principal"**
3. Digite sua legenda: `Transformador instalado em subestação`
4. Clique em **Salvar Alterações**

---

### ✅ Use "Legendas das Imagens (Array)" se:
- Você tem **VÁRIAS IMAGENS** (um carrossel que você pode navegar)
- Cada imagem precisa de sua própria legenda
- Exemplo: Você vê setas para passar de uma imagem para outra

**Como usar:**
1. Selecione a área (ex: Transporte)
2. Clique em **"Legendas das Imagens (Array)"**
3. Você verá uma caixa de texto grande
4. Digite **UMA LEGENDA POR LINHA**:
   ```
   Primeira imagem: Transformador de tração
   Segunda imagem: Vista interna dos enrolamentos
   Terceira imagem: Instalação em subestação
   ```
5. Cada linha = uma legenda para uma imagem (na ordem)
6. Clique em **Salvar Alterações**

---

## 🔍 Como Saber Qual Usar?

**Olhe na página da área:**
- Se você vê **setas** (← →) ou **pontos** embaixo da imagem = tem várias imagens = use **Array**
- Se você vê **apenas uma imagem** sem setas = use **Legenda da Imagem Principal**

---

## ❌ Problema: Não Consegue Salvar

### Verifique:
1. ✅ Você selecionou uma **Área** primeiro?
2. ✅ Você selecionou um **Campo** (Legenda)?
3. ✅ Você digitou algo no editor?
4. ✅ Qual mensagem de erro aparece? (copie aqui)

### Se aparecer erro:
- Me envie a mensagem de erro exata
- Me diga qual área você está editando
- Me diga qual campo você escolheu

---

## 💡 Dica Rápida

**Na dúvida, use "Legenda da Imagem Principal"** - é mais simples e funciona para a maioria dos casos!
