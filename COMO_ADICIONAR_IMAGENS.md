# Como Adicionar Imagens do Site Original

## 📥 Método 1: Download Manual (Recomendado)

### Passo 1: Acesse o Site Original
1. Abra o site original: https://zilmertransformado.wixsite.com/zilmer
2. Navegue até a seção "Transformadores para Instrumentos"

### Passo 2: Baixe as Imagens
1. **Clique com o botão direito** na imagem que deseja baixar
2. Selecione **"Salvar imagem como..."** ou **"Save image as..."**
3. Salve a imagem na pasta: `public/images/produtos/instrumentos/`

### Passo 3: Nomeie os Arquivos
Recomendações de nomes:
- `tp-interno.jpg` ou `tp-interno.png`
- `tp-externo.jpg` ou `tp-externo.png`
- `tc-interno.jpg` ou `tc-interno.png`
- `tc-externo.jpg` ou `tc-externo.png`

### Passo 4: Verifique o Formato
- Formatos aceitos: `.jpg`, `.jpeg`, `.png`, `.webp`
- Recomendado: JPEG para fotos, PNG para imagens com transparência

---

## 🖼️ Método 2: Usando Ferramentas do Navegador

### Google Chrome / Edge:
1. Pressione `F12` para abrir as Ferramentas de Desenvolvedor
2. Vá para a aba **"Network"** (Rede)
3. Filtre por **"Img"** (Imagens)
4. Recarregue a página (`F5`)
5. Procure as imagens dos transformadores na lista
6. Clique com botão direito na imagem desejada
7. Selecione **"Open in new tab"** (Abrir em nova aba)
8. Na nova aba, salve a imagem normalmente

---

## 📂 Estrutura de Pastas Criada

As seguintes pastas foram criadas para organizar as imagens:

```
public/
  images/
    produtos/
      instrumentos/    ← Coloque as imagens aqui
      oleo/           ← Para imagens de transformadores a óleo
      seco/           ← Para imagens de transformadores a seco
    projetos/         ← Para imagens de projetos
    clientes/         ← Para logos de clientes
    certificados/     ← Para imagens de certificados
```

## ✅ Após Adicionar as Imagens

Depois de adicionar as imagens, me avise e eu atualizarei o código para exibi-las automaticamente no site!

---

## 💡 Dica: Verificar URL da Imagem

Se você encontrar a URL direta da imagem no site original, pode me enviar e eu posso ajudar a baixá-la programaticamente.

**Exemplo de URL que você pode encontrar:**
- `https://static.wixstatic.com/media/xxxxx.jpg`
- URLs do tipo: `wixstatic.com` ou `wixsite.com`

## 🎨 Formatos Recomendados

- **JPEG (.jpg)**: Para fotografias de produtos
- **PNG (.png)**: Para imagens com fundo transparente
- **WebP (.webp)**: Melhor compressão (se disponível)
