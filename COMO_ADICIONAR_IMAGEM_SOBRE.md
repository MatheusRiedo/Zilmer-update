# 📸 Como Adicionar a Imagem na Página Sobre

## ✅ Estrutura Criada

A estrutura para exibir a imagem já está configurada na página "Sobre nós".

## 📁 Onde Colocar a Imagem

1. **Crie a pasta** (se ainda não existir):
   ```
   public/images/sobre/
   ```

2. **Nomeie o arquivo:**
   - `fabrica-sobre.jpg` (ou `.png`, `.webp`)
   - Ou renomeie o arquivo que você selecionou para este nome

3. **Coloque a imagem na pasta:**
   ```
   public/images/sobre/fabrica-sobre.jpg
   ```

## 🎨 Como Funciona

- A imagem será exibida entre o texto "Sobre nós" e os cards (Histórico, Clientes, Certificados)
- Layout responsivo e profissional
- Efeito de zoom suave ao passar o mouse
- Bordas arredondadas e sombra para destaque

## 📝 Se Quiser Usar Outro Nome

Se preferir usar outro nome de arquivo, edite:
- `app/sobre/page.tsx`
- Linha com: `src="/images/sobre/fabrica-sobre.jpg"`
- Altere para o nome do seu arquivo

## ✅ Após Adicionar

1. Coloque a imagem em: `public/images/sobre/fabrica-sobre.jpg`
2. Recarregue a página: http://localhost:3000/sobre
3. A imagem aparecerá automaticamente!
