# 📸 Onde Inserir Imagens - Guia Completo

## 📂 Estrutura de Pastas

As imagens devem ser colocadas nas seguintes pastas dentro de `public/images/`:

```
public/
  images/
    ├── certificados/     ← Certificados de qualidade, ISO, etc.
    ├── clientes/         ← Logos/logotipos dos clientes
    └── projetos/         ← Fotos de projetos realizados
```

---

## 🏢 **CLIENTES** (`public/images/clientes/`)

### O que colocar:
- **Logos/logotipos** das empresas clientes
- Formatos recomendados: `.png` (com fundo transparente) ou `.jpg`
- Tamanho recomendado: 200-400px de largura

### Como nomear:
- Use nomes descritivos, por exemplo:
  - `cliente-empresa-a.png`
  - `cliente-empresa-b.jpg`
  - `logo-petrobras.png`
  - `logo-vale.png`

### Exemplo de estrutura:
```
public/images/clientes/
  ├── cliente-1.png
  ├── cliente-2.png
  ├── cliente-3.jpg
  └── ...
```

**✅ Após adicionar:** As imagens aparecerão automaticamente na página `/sobre/clientes` em uma galeria de logos.

---

## 🏆 **CERTIFICADOS** (`public/images/certificados/`)

### O que colocar:
- **Fotos ou scans** de certificados de qualidade
- Certificados ISO, ABNT, etc.
- Formatos recomendados: `.jpg` ou `.png`
- Tamanho recomendado: 800-1200px de largura (para boa visualização)

### Como nomear:
- Use nomes descritivos, por exemplo:
  - `certificado-iso-9001.jpg`
  - `certificado-abnt.jpg`
  - `certificado-qualidade-2024.png`

### Exemplo de estrutura:
```
public/images/certificados/
  ├── certificado-iso-9001.jpg
  ├── certificado-abnt.jpg
  ├── certificado-qualidade.png
  └── ...
```

**✅ Após adicionar:** As imagens aparecerão automaticamente na página `/sobre/certificados` em uma galeria.

---

## 🏗️ **PROJETOS** (`public/images/projetos/`)

### O que colocar:
- **Fotos de projetos realizados**
- Transformadores instalados
- Obras em andamento ou concluídas
- Formatos recomendados: `.jpg` (melhor para fotos)
- Tamanho recomendado: 800-1200px de largura

### Como nomear:
- Use nomes descritivos, por exemplo:
  - `projeto-subestacao-2024.jpg`
  - `projeto-transformador-oleo.jpg`
  - `projeto-instalacao-industrial.jpg`

### Exemplo de estrutura:
```
public/images/projetos/
  ├── projeto-1.jpg
  ├── projeto-2.jpg
  ├── projeto-3.jpg
  └── ...
```

**✅ Após adicionar:** As imagens aparecerão automaticamente na seção "Projetos" da página inicial (`/`).

---

## 📝 **Passos para Adicionar Imagens:**

1. **Cole as imagens** nas pastas corretas:
   - `public/images/certificados/` para certificados
   - `public/images/clientes/` para logos de clientes
   - `public/images/projetos/` para fotos de projetos

2. **Nomeie os arquivos** de forma descritiva (sem espaços, use hífens)

3. **Formatos aceitos:** `.jpg`, `.jpeg`, `.png`, `.webp`

4. **Após adicionar:** As imagens aparecerão automaticamente no site!

---

## 💡 **Dicas:**

- ✅ **Otimize as imagens** antes de adicionar (reduza o tamanho do arquivo)
- ✅ Use **PNG com fundo transparente** para logos de clientes
- ✅ Use **JPG** para fotos de projetos e certificados
- ✅ Mantenha os **nomes dos arquivos em minúsculas** e sem espaços
- ✅ Evite nomes muito longos (máximo 50 caracteres)
